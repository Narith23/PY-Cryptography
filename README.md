## Getting Started

Py-Cryptography

### Prerequisites

- python 3.10 or Latest

### Installation

1. Creating virtual environments

   ```sh
   python -m venv /venv
   ```

2. Install Requirement packages

   ```sh
   pip install -r requirements.txt
   ```

### Start Server

- Run app

  ```sh
    python .\main.py
  ```
